# Contributing to ET NewsAction AI

Thank you for your interest in contributing to **ET NewsAction AI**! As a project built for the ET Hackathon 2026, we welcome community feedback, improvements, and bug fixes to help make business news universally actionable.

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project.
2. **Create a Branch**: Make your changes in a new branch (`git checkout -b feature/your-feature-name`).
3. **Run Locally**: Ensure both the FastAPI backend and React frontend run locally without errors.
4. **Commit Changes**: Use descriptive commit messages.
5. **Open a Pull Request**: Submit your PR with a clear description of what the PR solves or adds.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms—maintain respect and foster a welcoming, inclusive environment for all hackers and users.
